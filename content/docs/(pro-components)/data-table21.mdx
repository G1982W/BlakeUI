## Introduction

**Data Table 7** adds column management:

- **Header dropdown**: Each column header has a menu with sort ascending/descending and hide/show column.
- **Visibility**: Toggle column visibility from the header menu.
- **Sortable**: Sort from the same menu or by clicking the header.

Uses Table, Button, DropdownMenu, Badge, and TanStack Table.
