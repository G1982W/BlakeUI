## Introduction

**Data Table 8** supports resizable columns:

- **Resize handle**: Drag the vertical handle on the right edge of a header to change column width.
- **Sortable**: Click headers to sort.
- **Badges**: Availability shown as secondary badges.

Uses Table, Button, Badge, and TanStack Table.
