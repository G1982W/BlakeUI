## Introduction

**Data Table 5** is built for many rows:

- **Fixed height**: Scrollable body with sticky header.
- **Sortable columns**: Click headers to sort.
- **Row selection**: Checkbox column.
- **Row actions**: Three-dot menu per row (Edit, View, Copy ID).

For true virtualization (only visible rows in the DOM), add @tanstack/react-virtual and render rows in a virtualizer.

Uses Table, Button, Checkbox, DropdownMenu, Badge, and TanStack Table.
