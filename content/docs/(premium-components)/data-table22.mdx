## Introduction

Data Table 22 uses the same base [Table](/docs/components/table) component and [TanStack Table](https://tanstack.com/table/latest) as the standard [Data Table](/docs/premium-components/data-table). Use it when you need the same table capabilities with a dedicated doc for this variant.

## Installation

1. Add the base Table component and the Data Table component to your project.

2. Install `@tanstack/react-table`:

```bash
pnpm add @tanstack/react-table
```

<Installation
  dependencies={["@tanstack/react-table"]}

You also need the [Table](/docs/components/table) component in `@/components/ui/table.tsx`.

## Usage

```tsx
import { DataTable22 } from "@/components/ui/data-table22";

<DataTable22 />
```

For full documentation on columns, pagination, row actions, and extending with sorting and filtering, see the [Data Table](/docs/premium-components/data-table) doc.
