## Introduction

**Login Page 4** includes credentials and optional social login:

- **Logo and title**: Optional logo; configurable “Log in” heading.
- **Email + password**: Labeled inputs.
- **Remember me**: Checkbox and “Forgot password?” link.
- **Log in**: Primary submit button.
- **OR + social**: Optional Google, Facebook, and/or GitHub buttons (only shown if callbacks are passed).
- **Sign up**: Link at the bottom.

Uses Button, Input, and Checkbox.
