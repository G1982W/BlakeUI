## Overview

We don’t ship a dedicated Typography component. Instead, use Tailwind utility classes to build consistent headings, paragraphs, lists, and other rich text blocks.

This page is based on [shadcn/ui Typography](https://ui.shadcn.com/docs/components/radix/typography) and adapted for this codebase.

## Examples

```tsx
export function TypographyH1({ children }: { children: React.ReactNode }) {
  return (
    <h1 className="scroll-m-20 text-4xl font-extrabold tracking-tight text-balance">
      {children}
    </h1>
  );
}
```

## Notes

- These patterns are meant for UI surfaces (cards, modals, settings pages). Your docs pages may already apply their own “prose” styling.
- Feel free to extract these into local components (e.g. `TypographyH1`, `TypographyMuted`) if you want a shared API in your app.
