## Usage

Use the `banner()` helper to show a persistent notification (requires `<Toaster />` in your layout), or render `<Banner>` inline.

```tsx
import { banner } from "@/components/ui/banner";

banner("Your session will expire in 5 minutes.");
banner("Save your work before leaving.", { variant: "caution" });
banner("Verification failed.", { variant: "critical", title: "Error" });
```

```tsx
import { Banner } from "@/components/ui/banner";

<Banner variant="caution" onClose={() => setShow(false)}>
  Please review the updated terms.
</Banner>
```

## API Reference

### Banner

The Banner component renders a persistent alert with optional title, content, and close button.

<PropsTable component="Banner" />

### banner()

Show a persistent banner via the toast system. Banners do not auto-dismiss and display a close button.

```tsx
import { banner } from "@/components/ui/banner";

banner(message, { variant?, title?, description? });
```

| Option       | Type                    | Description                          |
| ------------ | ----------------------- | ------------------------------------ |
| `variant`    | `"default" \| "caution" \| "critical"` | Visual variant. Default: `"default"`. |
| `title`      | `React.ReactNode`       | Optional heading.                    |
| `description`| `React.ReactNode`       | Body content (overrides `message` when set). |
