## Usage

```tsx
import { Textarea } from "@/components/ui/textarea";
```

```tsx
<Textarea placeholder="Type your message here." />
```

The Textarea accepts all native `<textarea>` props, including `placeholder`, `disabled`, `rows`, `value`, and `onChange`.

## API Reference

### Textarea

The Textarea component displays a styled multi-line text input. It forwards all native textarea props and supports focus, invalid, and disabled styles.
