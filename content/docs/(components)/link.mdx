## API Reference

### AppLink

| Prop       | Type                       | Description                          |
| ---------- | -------------------------- | ------------------------------------ |
| `variant`  | `"primary" \| "secondary"` | Controls underline and color styles. |
| `disabled` | `boolean`                  | Disables interaction and focus.      |
| `href`     | `string`                   | Destination URL.                     |
